# 棋牌游戏记分器 V1
- 程序的功能：记录每轮游戏各玩家的分数（适用于棋牌类,进行多轮的游戏）,并进行结算
- 使用步骤：
  1.添加参与游戏玩家 
  2.每轮游戏结束后,选择-游戏记录-记录玩家得分
  3.游戏结束后,选择-游戏结算-,输出每位玩家的最终得分
